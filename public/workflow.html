배
배출처 (Emitter)
K-ITAD 전자
  • 생성/수정 배출관리 — 배출신청 5단계 등록, 수정/삭제, 예상정산금액
  • 조회 운송관리 — 실시간 Leaflet 지도 모니터링, 운송 알림
  • 조회 정산관리 — 자사 매입/매출 내역, 기간조회
  • 조회 자원순환 — 자사 순환율, 탄소절감, ESG 리포트
  • 조회 올바로연동 — 전자인계서 현황 확인
  • 관리 정보관리 / 설정
운
운송회사 (Transporter)
보안물류(주)
  • 생성/수정 배차관리 — 주간 캘린더 Drag&Drop, 기사/차량 배정
  • 조회 운송모니터링 — 실시간 지도, 봉인/GPS 추적
  • 조회 정산관리 — 운송비 매출 내역
  • 조회 올바로연동 — 전자인계서 현황
  • 관리 정보관리 / 설정
처
처리사 (Processor)
ITAD 처리센터
  • 승인 배출관리 — 배출요청 접수 확인, 상태 변경
  • 조회 운송관리 — 배차현황 조회, 운송 모니터링
  • 전체 데이터폐기 — 폐기 작업 등록, 검증, CoD 발급
  • 전체 자산처리 — 입고검수, 분해, 부품 추적, 출하
  • 관리 정산관리 — 전체 매입/매출, 세금계산서 발급
  • 조회 자원순환 — 전체 배출처 순환 성과, 회수가치
  • 제출 올바로연동 — 전자인계서 작성/제출
정
정부기관 (Government)
환경부
  • 조회 자원순환 — 전체 환경 성과 모니터링
  • 조회 설정
관
협회관리자 (Admin)
K-ITAD 협회
  • 전체 배출관리 — 전체 배출요청 통합 조회/관리
  • 전체 운송관리 — 배차현황 + 운송 모니터링
  • 전체 데이터폐기 — 전체 폐기 현황 조회
  • 전체 자산처리 — 전체 처리 현황 조회
  • 전체 정산관리 — 전체 정산 내역 통합 조회
  • 전체 자원순환 — 전체 배출처 순환 성과
  • 전체 올바로연동 — 전자인계서 통합 관리
  • 전체 설정 — 시스템 전체 설정
💰 정산 유형별 자금 흐름
정산유형 지급자 → 수취자 설명
자산매입금 처리사 → 배출처 자산 매각 대금
보안운송비 배출처 → 운송회사 보안 운송 서비스 비용
재사용 판매 유통사 → 처리사 리퍼비시 제품 판매 수익
파분쇄 파분쇄사 → 처리사 기판류 납품 수익
재활용원료 제련사 → 처리사 금속류(구리/금/은) 회수 수익
확인서 발급비 배출처 → 처리사 CoD 인증서 발급 수수료
